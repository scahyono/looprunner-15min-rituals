# LoopRunner: 15-Minute Rituals

LoopRunner is a minimalist single-page life loop game. Each loop is a bite-sized ritual (health, wealth, spirit) that you can start, track, and streak through. Finish a loop to level up your daily dashboard and keep the momentum going.

## How to Run
1. Clone or download this repository.
2. Open `index.html` directly in any modern web browser.

No build process or dependencies are required.
